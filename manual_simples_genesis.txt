
MANUAL SIMPLES – EXECUÇÃO DO PROTÓTIPO GENESIS (Streamlit)

1) Requisitos:
- Python 3 instalado
- Pacotes necessários:
    py -m pip install streamlit plotly pandas numpy


2) Arquivo principal:
- genesis_app.py

3) Como executar:
1. Abra o PowerShell na pasta onde está o arquivo:
   - Vá até a pasta
   - Clique na barra de endereço
   - Digite: powershell
   - Aperte ENTER

2. Execute o aplicativo digitando:
   py -m streamlit run genesis_app.py


3. O navegador abrirá automaticamente.
   Se não abrir, acesse o link mostrado (exemplo):
   http://localhost:8501

4) Como parar:
No PowerShell, pressione:
CTRL + C

Pronto. O protótipo estará funcionando com as abas:
• REVEAL – abandono por etapa e canal
• PREDICT – risco de desistência
• ACT – ações sugeridas
